Theme & language

Set how Vehium Workshop looks and what language it speaks.

Theme

Three options:

  • System — follows your device’s appearance setting. This is the default.
  • Light — always light mode.
  • Dark — always dark mode (easier on the eyes in a dim workshop).

Your choice is stored per device. Switching themes on your phone doesn’t change the web or desktop app.

UI only

The language setting changes only the Vehium interface (menus, buttons, labels). Data you’ve entered — client names, service descriptions, invoice line items — stays exactly as you typed it.

How to change

  1. Open Settings → Appearance.
  2. Pick a theme tile (System / Light / Dark).
  3. Pick a language from the dropdown. Each option shows its flag.
  4. The app switches immediately — no save button, no reload.

Don’t see your language?

We prioritize new languages by user demand. If yours is missing, email info@vehium.com — requests are reviewed and tracked.
