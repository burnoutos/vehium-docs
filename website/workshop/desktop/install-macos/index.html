Install on macOS

Runs on Apple Silicon and Intel Macs.

Requirements

  • macOS 11 Big Sur or later.
  • Approximately 350 MB free disk space.
  • An internet connection for sign-in.

Download

  1. Open vehium.com/#download.
  2. Click Download for macOS. The file is a signed .dmg (~120 MB).

Install

  1. Double-click the downloaded DMG.
  2. Drag the Vehium Workshop app icon into the Applications folder.
  3. Eject the DMG (click the eject icon next to it in Finder).

First launch

  1. Open Applications and double-click Vehium Workshop.
  2. macOS may show “‘Vehium Workshop’ is an app downloaded from the Internet” — click Open. This prompt appears only on the first launch.
  3. Sign in with your Vehium username and password.
Gatekeeper

The app is signed and notarized, so Gatekeeper won’t block it. If you see a hard “cannot be opened” error, the download may have been truncated — re-download the DMG.

Updates

Same auto-update behaviour as the Windows build: check on launch, install in the background, prompt to restart.

Uninstall

Drag Vehium Workshop from Applications to the Trash. Empty the Trash to complete removal. User preferences remain in ~/Library/Application Support/Vehium; delete that folder to wipe everything.
