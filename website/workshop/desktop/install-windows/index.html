Install on Windows

Takes about a minute on a modern PC.

Requirements

  • Windows 10 (64-bit) version 1809 or later, or Windows 11.
  • Approximately 300 MB free disk space.
  • An internet connection for sign-in (offline use after that is limited).
  • Admin rights on the PC to run the installer.

Download

  1. Open vehium.com/#download.
  2. Click Download for Windows. The installer is an NSIS .exe (~100 MB).

Install

  1. Double-click the downloaded file (e.g. Vehium Workshop Setup 3.0.6.exe).
  2. SmartScreen may warn “Windows protected your PC” — click More info → Run anyway. The installer is code-signed; SmartScreen calms down once it has seen enough installs.
  3. Choose install scope: only me (no admin) or all users (admin required).
  4. Accept the default install path or pick your own.
  5. Click Install and wait a few seconds.
  6. Tick Launch Vehium Workshop and click Finish.

First launch

On first open, the app window appears with the sign-in screen. Log in with your Vehium username and password — the app remembers you on subsequent launches.

Auto-start on boot

In the app, open Settings → Desktop (only visible in the desktop build) and toggle Start Vehium when Windows boots.

Updates

The app checks for updates on every launch and installs them in the background. You’ll see an update-ready banner when a new version is downloaded; click it to restart into the new version.

Uninstall

  1. Open Settings → Apps → Installed apps.
  2. Find Vehium Workshop.
  3. Click Uninstall and confirm.
