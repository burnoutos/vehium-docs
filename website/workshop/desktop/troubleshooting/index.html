Desktop troubleshooting

Fixes for the most common issues.

Installer won’t run

  • SmartScreen warning (Windows): click More info → Run anyway. The installer is signed.
  • Gatekeeper block (macOS): the DMG may be corrupted — re-download. If it still fails, right-click the app → Open to bypass the warning.
  • Not enough disk space: free up 500 MB and try again.

App opens with a blank window

  • Check your internet — the desktop app needs connectivity to sign in.
  • Close the app, reopen it. If still blank after 30 seconds, restart your PC.
  • If a corporate firewall blocks outbound TLS to *.vehium.com, whitelist those hosts.

Auto-update fails

  • Quit and relaunch the app to retry.
  • If it keeps failing, download the fresh installer from vehium.com/#download and reinstall over the top — your data is in the cloud, nothing is lost.

High CPU usage

A single Vehium window should sit well under 5% CPU at idle. If you see high CPU:

  • Check the dashboard auto-refresh interval — if network is flaky, retries can spike CPU.
  • Try closing other Electron apps (Slack, VS Code, Teams) — GPU contention can surface on older Macs.
  • Update to the latest version.

Grabbing logs for support

If you email info@vehium.com, attach the app log. Find it at:

  • Windows: %APPDATA%\Vehium\logs\main.log
  • macOS: ~/Library/Application Support/Vehium/logs/main.log
