Create a service

Open a new work order. Everything else — inspection items, photos, invoice — layers on afterwards.

Open the dialog

  • Services page → + New service in the top-right.
  • Dashboard → New service quick action.
  • Vehicle detail page → + New service in the Service history section (vehicle pre-filled).

Fields

FieldRequiredNotes
VehicleYesSearchable by plate, VIN, make, model, year. If no match, click Create vehicle to add one inline.
DateYesdd/mm/yyyy in locale-appropriate order. Cannot be earlier than the vehicle’s last service date.
OdometerOptionalMust be ≥ the last recorded odometer for this vehicle. Warns if you type less.
StatusYesDefault is In progress. See Service statuses.
DescriptionOptionalFree text. Shown on the service card and on invoices.
Assigned employeeOptionalWho is doing the work. Defaults to the creator.
Save first, add items next

You need to save the service before adding inspection items or photos. Don’t worry about filling in every detail at creation time — a service is a record you build up over the job.

After saving

You land on the service detail page with empty Inspection items, Photos, and Invoice sections ready to populate. The service also appears on:

  • The main Services list (in the In progress tab).
  • The vehicle’s Service history section.
  • Every linked client’s Service history section.
  • The dashboard activity feed.
