Add a client

Create a new client record. Takes about 30 seconds.

How to open the form

Three ways:

  • Clients page → click + New client in the top-right.
  • Dashboard → click the New client quick action.
  • Service creation → in the vehicle picker, click New client to create one inline.

Fields

FieldRequiredNotes
First nameYesUp to 100 characters.
Last nameYesUp to 100 characters.
EmailRecommendedEnables linking to the Vehium customer app. Must be valid email format if provided.
PhoneRecommendedInternational format supported. A country-code selector is shown next to the number field.
Street addressOptionalShown on invoices.
CityOptional 
State / provinceOptional 
Postal codeOptional 
CountryOptionalDropdown, defaults to your workshop’s country.
Email first, vehicle second

Fill in the client’s email before adding their vehicle. Vehium uses the email to link the vehicle to the customer app, so alerts (like MOT reminders) appear on the vehicle owner’s phone.

After you save

The client appears immediately in the Clients list. The form closes automatically; you can click Save & add another if you have a batch of clients to enter.

Common validation errors

  • Email already used — another client has the same email. Search first, then edit rather than duplicating.
  • Invalid phone — the number doesn’t match the selected country’s format. Try adjusting the country code.
  • Missing required field — first or last name is blank.
